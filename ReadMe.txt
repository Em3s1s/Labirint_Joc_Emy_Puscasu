Proiectul este dedicat sistemelor de tip Linux;

Pentru rularea proiectului pe Windows trebuie instalata aplicatia 'Ubuntu' din 'Microsoft Store';

Acest proiect poate fi compilat si rulat cu ajutorul aplicatiei Code::Blocks (acesta poate fi instalat direct din Software Manager) sau cu ajutorul consolei 'Terminal' specifice sistemului de operare Linux.

Pentru a accesa Terminal-ul apasati combinatia de taste 'Ctrol'+'Alt'+'T'.

Dupa ce s-a deschis fereastra Terminalului tastati urmatoarele comenzi:

-'sudo apt update'
-'sudo apt install build-essential'
-'cd <calea/proiectului>'    
    ex."cd Desktop/Proiecte/Labirint"(pt Linux) sau "cd mnt/c/proiecte" (pt Windows daca salvati proiectul pe partitia 'c')
-'make' 
-'./Maze'
